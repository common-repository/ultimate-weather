Ultimate Weather helps you to display weather information on your wordpress website.

== Description ==

Demo: http://clownfishweb.com/demo/ultimate-weather/


= Ultimate Weather Features =

- Works with any WordPress theme.
- Fully Responsive.
- Can Display weather information upto 10 days.
- Nice sleek design.
- Friendly Support and fully documented.

== Installation ==

1. Upload the plugin files to the '/wp-content/plugins/PLUGIN_NAME' directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
